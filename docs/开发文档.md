# 0

lender operating country应该是基本正确的，但是lender institution type里面有不少不准确

# 1 根据后缀删除非银行组织

规则：
- 若**以**特定关键词（`fund`, `investment corp`, `llc`, `l.l.c.`，`LLC`）结尾：
  - 若`Lender_Institution_Type`**非空**：
    - 若里面有 `Bank` , 不排除
    - 若里面没有 `Bank` , 排除
  - 若`Lender_Institution_Type`**为空**，不排除
- 若**不以**特定关键词（`fund`, `investment corp`, `llc`, `l.l.c.`，`LLC`）结尾，不排除



# 2 过滤非美国公司

规则：

-  Lender_Institution_Type 中包含 "US Bank" 的，那就认为是美国的
- Lender_Operating_Country 中为 `United States` 的



# 3 AI判断是否为美国银行

**分类标准**：

- **“是美国银行”**：指该实体是总部在美国、持有美国联邦或州银行牌照的**商业银行、储蓄银行或全国性银行**（National Association）。这包括了外国银行在美国设立的、独立持牌的银行子公司（如 TD Bank NA, Santander Bank NA）。
- **“不是美国银行”**：包括以下几类：
  - 总部在其他国家的**外国银行**（如 Barclays, HSBC, BNP Paribas）。
  - **非银行金融机构**，如投资公司、资产管理公司、私募股权基金、保险公司等（如 BlackRock, Ares Capital, Morgan Stanley 的非银行实体）。
  - **多边开发金融机构**（如 World Bank, IFC）。
  - **美国农业信贷体系**（Farm Credit System）的成员，因其特殊的合作金融性质，未归入传统商业银行类别。

基准以Gemini得出，多次查询补充检查
